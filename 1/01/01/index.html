###Resume Makeover - Do it around May/June For Impressing Recruiters/Getting shortlisted

###LinkedIn Profile - Do it ASAP. Connect with as many people as possible. Join Groups for jobs. For Impressing Recruiters and for contacts. Get more referrers.

###Apply to Companies/Through Referrers - Do it after Reume Makeover First try through referrers - reach out to people in TCS/Wipro/Accenture etc. and ask them to refer you as fresher. Referrals have more chance of getting selected.

###Contribute on Github - Start it today. Make it a plan to contribute to a simple Java Project. Highly recommended. This will boost your confidence and impress your interviewer like never before. I’ll also tell you to contribute to my open source projects.

###Create Simple Apps - Start Today. Create apps and games, etc. and upload on Github. Shows you know what you are doing and are interested in Open Source. For Java, try to create - A Web server - Chat Serve and Client

###Medium Priority - Start today. plan out so that you can complete these before May Computer Science Basics. Helps in interviews and in the long Run. Optional But highly recommended. http://ocw.mit.edu/courses/electrical-engineering-and-computer-science/6-042j-mathematics-for-computer-science-fall-2010/video-lectures/ https://www.udacity.com/courses/cs101 https://www.coursera.org/course/cs101 https://www.udacity.com/course/software-testing--cs258

###High Priority - Must for Java - Start Today These will boost your java skills like anything. Complete all the assignments. Try to get 100% score. https://www.coursera.org/course/algs4partI https://www.coursera.org/course/algs4partII Read “Effective Java” - Attached https://www.udemy.com/java-multithreading/ Read “Java Concurrency in practice” - Attached Reac “Cracking the Coding Interview” - Attached

###Web Technologies - A must - Start today Web technologies are used everywhere. These will teach you html/CSS/javascript https://www.codecademy.com/learn/web https://www.codecademy.com/learn/javascript https://www.codecademy.com/learn/jquery

###Git - Must. A Basic - Start asap Git is a must. Learn how to use source control. Even for simple apps that you create. https://www.codecademy.com/learn/learn-git

####Java Web Technologies - Start asap These are technologies used in TCS/Wipro/Accenture. Learn these technologies if you want to land a job in these companies. https://www.codecademy.com/learn/learn-sql https://www.codecademy.com/learn/learn-java http://www.tutorialspoint.com/webservices/ http://www.tutorialspoint.com/restful/ http://www.tutorialspoint.com/jpa/ http://www.tutorialspoint.com/spring/ http://www.tutorialspoint.com/hibernate/

Linux/Command Line/Bash

You will have to use Linux/Unix some day or other. Learn this. Thank me later. https://www.codecademy.com/learn/learn-the-command-line http://www.learnshell.org/

###High Priority Improve your coding skills so that you don’t get stuck during interviews. - Solve around 10 problems from topcoder.com - Write programs on whiteboard in front of a friend if possible.
