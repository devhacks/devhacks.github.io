Freshers' guide to land their dream job in java

2016-01-16 · 581 words · 3 minute read
programming

Today, I got a call from a friend’s sister in final year of Computer Science Engineering. Like many other engineering students in India, she has the same concern - ‘How can I get a job? There are very few campus interviews in my college.’. Now, if you are one of those few lucky guys who studied in a tier 1 or tier 2 engineering institute, you might not worry too much. But there are lots of engineering students in India who are struggling to get a job after graduation. So, I asked for her resume and tt turns out she doesn’t know anything which can be useful in a work environment. And it’s not her fault. Almost every fresher I have met have very little practical knowledge. So, I sent her a list of “to do” tasks for the next 4 months as she has around 4 months (not counting couple of months in consideration of their exams) till the end of her degree. I borrowed some from Google’s Guide to Students and added a few extra tasks for Java since she wants to work on java technologies. It’s a long list but considering the fact that I have covered each item and more in last few months, I hope it’s not too much. So, here’s the list.

Resume Makeover - (after 4 months)

  • Do it around May/June
  • For Impressing Recruiters/Getting shortlisted

LinkedIn Profile - Do it right away.

  • Connect with as many people as possible. Join Groups for jobs.
  • For Impressing Recruiters and for contacts. To get more referrers and job opportunities

Apply to Companies - After Resume Makeover

  • First try through referrals - reach out to people in TCS/Wipro/Accenture etc. and ask them to refer you as fresher. Referrals have more chance of getting shortlisted.

Contribute on Github - Start Today

  • Make it a plan to contribute to a simple Java Project.
  • Highly recommended. You will learn a lot, become more confident and it will impress your interviewer like never before.

Create Simple Apps - Start Today.

  • Create apps and games, etc. and upload on Github.

For Java, try to create

  • A Web server
  • Chat Serve and Client
  • An Android App

CS Basics - Start today

  • Plan out so that you can complete these before May
  • Computer Science Basics. Helps in interviews and in the long Run. Optional but highly recommended.
  • http://ocw.mit.edu/courses/electrical-engineering-and-computer-science/6-042j-mathematics-for-computer-science-fall-2010/video-lectures/
  • https://www.udacity.com/courses/cs101
  • https://www.coursera.org/course/cs101
  • https://www.udacity.com/course/software-testing--cs258

Dive Deeper into Java - Start Today

  • These will boost your java skills like anything. Complete all the assignments. Try to get 100% score.
  • https://www.coursera.org/course/algs4partI
  • https://www.coursera.org/course/algs4partII
  • Read “Effective Java”
  • https://www.udemy.com/java-multithreading/
  • Read “Java Concurrency in practice”
  • Reac “Cracking the Coding Interview”

Web Technologies - A must - Start today

  • Web technologies are omnipresent. These will teach you html/CSS/javascript.
  • https://www.codecademy.com/learn/web
  • https://www.codecademy.com/learn/javascript
  • https://www.codecademy.com/learn/jquery

Git - Use it in your own projects

Git is a must. Learn how to use source control. Even for simple apps that you create. Indispensible.

  • https://www.codecademy.com/learn/learn-git

Java Web Technologies - Start asap

These are technologies used in TCS/Wipro/Accenture. Learn these technologies if you want to land a job in these companies.

  • https://www.codecademy.com/learn/learn-sql
  • https://www.codecademy.com/learn/learn-java
  • http://www.tutorialspoint.com/webservices/
  • http://www.tutorialspoint.com/restful/
  • http://www.tutorialspoint.com/jpa/
  • http://www.tutorialspoint.com/spring/
  • http://www.tutorialspoint.com/hibernate/

Linux/Command Line/Bash

You will have to use Linux/Unix some day or other. Learn this. Thank me later.

  • https://www.codecademy.com/learn/learn-the-command-line
  • http://www.learnshell.org/

Coding Practice

Improve your coding skills so that you don’t get stuck during interviews.

  • Solve around 10 problems from topcoder.com
  • Write programs on whiteboard in front of a friend if possible.

Well, that’s it. Let me know in comments if there’s anything else you would suggest.
